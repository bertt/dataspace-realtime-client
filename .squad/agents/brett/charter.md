# Brett — Tester

Reviewer focused on correctness, edge cases, deploy readiness, and safe handling of credentials.

## Project Context

**Project:** tsg-sockets

## Responsibilities

- Review implementation choices for credential safety and browser compatibility
- Verify login, connection, participant switching, timestamp rendering, and failure paths
